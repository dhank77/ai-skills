# AI Skills

Custom skills for the Kilo AI CLI. This repository is intended to grow into a collection of AI skills covering the tools, workflows, and conventions needed for everyday software engineering, starting with Laravel-focused workflows.

## Current Skills

| Skill | Short Description |
|-------|-------------------|
| `skils/stitch-laravel-converter` | Convert raw HTML/Tailwind into reusable Blade layouts, components, and a production-ready asset setup. |
| `skils/stitch-laravel-functionalizer` | Connect static Blade views to Laravel backend pieces: models, migrations, routes, controllers, forms, validation, and dynamic data. |

## Planned Areas

Additional skills may cover topics such as:
- Backend architecture and API design
- Frontend frameworks and component patterns
- Testing, CI/CD, and deployment
- Database design and optimization
- Developer workflow and tooling

## How To Use

1. Ensure Kilo loads skills from this repository.
2. Use the appropriate skill for the task at hand.
3. Follow each skill's `SKILL.md` and bundled references before implementing.
4. Add new skills under `skils/<skill-name>/` following the existing structure.

## Notes

- Skills should follow the project's general standards: DRY, accessibility, and framework conventions where applicable.
- For repo-level Kilo configuration, see `AGENTS.md` and `kilo.json`.
